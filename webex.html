Webex Calling FAQs

Everything you need to know about Webex Calling

Cloud Connect is a shared SIP integration with Certified Cloud Connect providers. Webex Calling customers can work directly with a Certified provider of their choice. Then, users can select their Cloud Connected PSTN provider to route calls to the PSTN in the Webex Control Hub.

Yes, Cloud Connect also works for your Webex Contact Center. BRCK is a Certified Calling Provider, so we can power both your Webex Calling and Contact Center systems.

Local Gateway for Webex Calling is a premises-based way to deliver PSTN connectivity. Local Gateway also provides connectivity to on-premises IP PBX deployments, like Cisco Unified Cloud Manager (CUCM).

Many organizations are moving from CUCM to Webex Calling for a better user experience, cloud calling, and mobility. BRCK offers PSTN connectivity for CUCM and Webex Calling, so you can connect now and migrate when you’re ready.

Webex Calling and Zoom are both cloud-based phone systems that offer Unified Communications and VoIP services. Both platforms offer toll-free numbers, voicemail transcription, video and audio conferencing, scheduling, authentication, waiting rooms, and recording.
